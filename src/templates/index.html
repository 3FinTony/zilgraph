Zilgraph

Objective

Zilgraph is an open source tool for visualizing zilswap activity on the Zilliqa blockchain.

It's providing XSGD and GZIL live charts.

The data is acquired from the zilswap smart contract and stored in a local MongoDB database.

The Zilgraph front-end provides live visualized data using flask and chartjs.
